WNG Numeric Sequence App

This application presents 5 different numeric sequences with values starting from 1 up to (and including) the number sepcified in the input box. All input values must be whole (non decimal) integers greater than 0, with 1 bieng the minimum acceptable value. The first sequence displays all the values starting from 1 to the specified value. The next two sequences display the odd and even numbers in the range from 1 up to the specified value. The next sequence displays all values from 1 up to the specified value but replaces multiples of 3, 5 and both 3 & 5 with C, E and Z respectively. The final sequence will output the Fibonacci sequence up to the value sepcified. Note that this sequence will start with a single 1 as this is the minimum value. Once you press the Calculate button the previously described sequences will get populated below as comma seperated lists or an error message will be displayed in red.
